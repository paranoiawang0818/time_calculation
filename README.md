# ⏰ 时间计算器

一个简单实用的时间计算器。  
输入开始时间和结束时间，自动计算并输出两者之间的时长。  

## ✨ 功能介绍

- 输入 **开始时间**
- 输入 **结束时间**
- 自动计算时间差
- 输出对应的 **时长结果**

适合用于：

- 记录学习时长 📚
- 统计工作时间 💼
- 计算活动持续时间 🎯
- 日常时间管理 ⌛

## 🛠️ 使用方式

1. 输入开始时间
2. 输入结束时间
3. 点击计算 / 运行程序
4. 查看输出的时长结果

## 🧾 输入示例
开始时间：09:30
结束时间：11:45
## ✅ 输出示例
时长：2小时15分钟

## 🚀 项目特点
- 功能简单，使用直观
- 适合初学者练手

## 📂 项目目标
通过这个小项目，实现最基础的时间差计算功能，帮助用户快速得到两个时间点之间的持续时长。
